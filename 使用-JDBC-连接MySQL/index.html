使用 JDBC 连接MySQL

发表于 2018-06-18 | 分类于 Java

JDBC（Java Database Connectivity，Java数据库连接）是 Java 语言中用来规范客户端程序如何来访问数据库的应用程序接口，提供了诸如查询和更新数据库中数据的方法。
本文讲述如何使用 JDBC 来连接和访问数据库。

为方便引入 JDBC 依赖包，我们创建 Maven 项目来实现我们的示例程序。
